Customizing the book cover

In this chapter you will learn to set your own cover image to replace the default one. To do this you are going to replace the default cover image file with your own one.

Choose an image that you would like to use as the frontpage image. Images with a resolution of 768x1024 work best.

Now you should open up your FTP-client and replace the cover.jpg file in the images/ folder of the Cookbook theme. This file is probably located at this location: wp-content/themes/cookbook/images/cover.jpg.

Refresh the page on your Desktop or on your iPad and your new cover image should be there!

Choosing posts or pages

The WordPress Cookbook theme supports two different settings for displaying content:

  • You can choose to show all the posts in your WordPress installation just like on a regular blog. Each blog post will have its own swipable page.
  • Or you can choose to display all WordPress pages.

To configure this setting, go to Settings > Reading and choose either Your latest posts or A static page.

When you choose A static page you can select any arbitrary page from the dropdowns.

When the Your latest posts option is enabled the theme will load your latest posts.
If the A static page option is enabled. The theme will load all the pages you have created.

Adding text and writing a recipe (shortcodes)

Now let's add some content!

You can add content to your pages, just like you would normally do on any WordPress site. Just start typing your text into the editor, save your page and it all works.

We have however added two shortcodes that should allow you to really simply replicate the beautiful look & feel of a paper cookbook.

It's better to show you, so here is an example:

[introduction]

This is my favorite dish that my mom makes when there is a party. It's really simple and you only need some potatoes, olives and some pork.

[/introduction]

[columns]

Ingredients (h2)
  • 2 potatoes
  • 200 grams pork
  • olives

[/columns]

Shortcode: [introduction]

You can use this shortcode to add an introduction paragraph at the top of the page, or when you are starting to write in columns.

Shortcode: [columns]

On webkit-enabled browsers (like Chrome, Safari and the iPad) this shortcode wil automatically arrange the text inside it into a beautiful 2-column layout.

It will automatically fit your text to the columns and it makes sure that the columns will never break inside a paragraph. This way, visitors will get a better reading experience.

Adding a recipe header image with Featured post images

Finally, let's add a beautiful image as a header for a recipe. A header image is a great opportunity to show off how the end result of the dish you are cooking will look like.

Adding such an image is easy. Just add it as a Featured image to your page or post by using the sidebar widget. As soon as you set the featured image and publish the header will appear on the page.
